🌿 AyurBazaar

AyurBazaar is a simple e-commerce website built using Flask (Python), HTML, CSS, and SQLite.

It focuses on Ayurvedic and natural products like medicines, cosmetics, herbal tea, and grocery items.

🚀 Features

- 👤 User Registration & Login
- 🛒 Add to Cart with Quantity
- ❌ Remove from Cart
- 💰 Total Price Calculation
- 💳 Payment Page (Demo)
- 📦 Order Placement & History
- 🛠️ Admin Panel (Add Products)
- 🖼️ Product Image Upload
- 🔍 Search & Category Filter

🛠️ Tech Stack

- Python (Flask)
- HTML, CSS
- SQLite Database

📂 Project Structure

- app.py → Main backend
- templates/ → HTML files
- static/ → CSS & images
- database.db → Database

🌐 Future Improvements

- Real Payment Gateway (Razorpay)
- Cloud Image Storage
- Live Order Tracking
